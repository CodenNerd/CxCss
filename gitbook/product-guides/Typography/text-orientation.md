### Text Orientation Classes

#### `.text-orientation-mixed`

Sets the text orientation to mixed.

CSS equivalent: `text-orientation: mixed;`

Example usage:
```html
<div class="text-orientation-mixed">
  <p>This text has a mixed text orientation.</p>
</div>
```

---
